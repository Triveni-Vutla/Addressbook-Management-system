The Address Book Management System is a simple application used to store and manage contact details such as name, phone number, and email.
It allows users to add, view, search, update, and delete contacts easily. The data is stored using files, so information remains saved even
after closing the program. The system is designed to be easy to use and efficient for handling contact records.

The project is implemented using the C programming language and uses concepts like structures, functions, and file handling. Structures are used
to store contact details, functions are used to perform operations, and file handling ensures data is stored permanently. Basic input and output operations
are used for user interaction.

This system can be used for managing personal contacts, maintaining small business records, and organizing directories. It also helps in understanding 
core programming concepts and can be extended with advanced features like databases or user interfaces.
